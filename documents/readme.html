Instructions to compile:

  1. Install ant in your machine

  2. Download the jar file into your machine

  3. Unjar it by executing the following command : 
         $> jar xvf <jarfilename>
  4. Enter into directory ClusterInfrastructure 
         $> cd ClusterInfrastructure
  5. Set your JAVA_HOME and ANT_HOME environment variables

  6. Execute the following command to compile the code :
         $> ant

Instructions to run Compute Space in remote machine:

  1. SSH to remote machine by executing the following command : 
         $> ssh -X <username@hostname>
  2. Enter your Password

  3. Make sure compilation step described in "Instructions to Compile" is completed

  4. Execute the following command to start the Compute Space:
         $> ant runSpace

Instructions to run n number of remote computers

  1. SSH to remote machine by executing the following command : 
         $> ssh -X <username@hostname>
  2. Enter your Password

  3. Make sure compilation step described in "Instructions to Compile" is completed

  4. Change the hostname in build.xml to the remote machine name which runs the Compute Space

  5. Also Change the hostnames in build.xml to the remote machine name which runs the Compute Engine

  6. Execute the following command to start n computers in remote machines :
         $> ant runComputer

Instructions to run Clients in remote machine

  1. SSH to remote machine by executing the following command : 
         $> ssh -X <username@hostname>
  2. Enter your Password

  3. Make sure compilation step described in "Instructions to Compile" is completed

  4. Change the hostname in build.xml to the remote machine name which runs the Compute Space

  5. Execute the following command to start a client to execute Travelling Salesman Problem and display results :
         $> ant runEuclideanTspClient
  6. Execute the following command to start a client to execute MandelbrotSet Problem and display results :
         $> ant runMandelbrotSetClient
  7. Execute the following command to start a client to execute Fibonacci Problem and display results :
         $> ant runFibonacciClient
